Advent of Code 2025

https://adventofcode.com

Implemented in Racket and Rust


----------
Day One:
----------
Rotate a dial labeled 0-99, and track the number of times it lands on position 0


----------
Day Two:
----------
Track and accumulate invalid IDs in a list of integer ranges.
Invalid IDs are integers where it is a repeated sequence of digits, e.g., 55, 6464, and 171171.


---------
Day Three:
---------
Find the largest subset of digits for a set of large numbers.


---------
Day Four:
---------
Find set of characters bordered by a certain amount of the same characters.


--------
Day Five:
--------
Sort and count IDs within specified ranges, which may be overlapping.


--------
Day Six:
--------
Reduce columns of input to sums or products based on operator at the bottom of the columns.
