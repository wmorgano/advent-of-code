Advent of Code 2025

https://adventofcode.com

Implemented in Racket and Rust


----------
Day One:
----------
Rotate a dial labeled 0-99, and track the number of times it lands on position 0


----------
Day Two:
----------
Track and accumulate invalid IDs in a list of integer ranges.
Invalid IDs are integers where it is a repeated sequence of digits, e.g., 55, 6464, and 171171.
